Note

Piexif supports Python versions 2.7, 3.3, 3.4, Pypy, Pypy3

‘easy_install’:

$ easy_install piexif

or ‘pip’:

$ pip install --pre piexif

or download .zip, extract it and run:

$ python setup.py install
